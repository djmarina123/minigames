# Hospedar a Política de Privacidade — MiniPlay

A Google Play exige uma **URL pública e acessível** (não arquivo local, não link privado).

Arquivos prontos no repositório:

| Arquivo | Uso |
|---|---|
| [`privacy-policy.md`](privacy-policy.md) | Texto completo (fonte) |
| [`privacy-policy.html`](privacy-policy.html) | Página HTML para hospedagem |

**Contato configurado:** `djmarina@gmail.com`

---

## Opção 1 — GitHub Pages (recomendada, grátis)

Você já tem o repo em `github.com/djmarina123/minigames`.

1. Substitua o e-mail nos arquivos acima
2. Faça commit e push
3. No GitHub: **Settings → Pages**
4. **Source:** Deploy from a branch
5. **Branch:** `main` → pasta **`/docs`**
6. Salvar — aguarde 1–5 minutos

URL final:

```
https://djmarina123.github.io/minigames/privacy-policy.html
```

Cole essa URL na Play Console → **Política do app → Política de privacidade**.

---

## Opção 2 — Google Sites (grátis, sem código)

1. Acesse [sites.google.com](https://sites.google.com)
2. Crie site em branco
3. Copie o texto de `privacy-policy.md` para a página
4. Publique → **Publicar na Web**
5. Use a URL gerada (ex.: `https://sites.google.com/view/miniplay-privacidade`)

---

## Opção 3 — Notion (página pública)

1. Cole o conteúdo em uma página Notion
2. **Share → Publish to web**
3. Use o link público

---

## O que NÃO funciona

- Arquivo só no computador
- Link do Google Drive **privado**
- Screenshot ou PDF sem URL
- `file:///` ou caminho local
